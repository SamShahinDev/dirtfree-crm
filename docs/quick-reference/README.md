## Printing Guidelines

### PDF Conversion

Convert markdown files to PDF using one of these methods:

**Method 1: Pandoc (Recommended)**
```bash
pandoc opportunities-cheatsheet.md -o opportunities-cheatsheet.pdf --pdf-engine=wkhtmltopdf -V geometry:margin=0.5in
```

**Method 2: Markdown to PDF (VS Code Extension)**
1. Install "Markdown PDF" extension in VS Code
2. Open cheatsheet file
3. Right-click > "Markdown PDF: Export (pdf)"

**Method 3: Online Converter**
- Use https://www.markdowntopdf.com/
- Upload markdown file
- Download PDF

### Print Settings

For best results:
- **Paper Size**: US Letter (8.5" x 11")
- **Orientation**: Portrait
- **Margins**: 0.5 inches all sides
- **Color**: Full color recommended (works in B&W too)
- **Quality**: High quality / Best

### Recommended Paper

- **Standard Office**: Regular copy paper (20lb)
- **Durable Reference**: Cardstock (65lb-110lb)
- **Laminated**: Print on cardstock, then laminate for durability
