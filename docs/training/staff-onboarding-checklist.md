# Staff Onboarding Training Checklist

**Dirt Free CRM** | Complete Training Program for New Team Members

---

## How to Use This Checklist

**For Managers:**
- Assign this checklist to all new hires
- Schedule dedicated training time
- Pair new employee with mentor
- Track completion progress
- Test knowledge before signing off

**For New Employees:**
- Work through each section in order
- Check off items as you complete them
- Ask questions when unclear
- Practice in test/demo environment first
- Don't skip ahead - each section builds on previous

**Timeline**: Complete within first 2 weeks

---

## PRE-TRAINING SETUP

**Before Day 1, Manager Completes:**

- [ ] User account created in CRM
- [ ] Correct role and permissions assigned
- [ ] Email credentials sent to new hire
- [ ] Portal access verified
- [ ] Test environment access (if available)
- [ ] Training materials gathered (manual, videos, cheatsheets)
- [ ] Mentor/trainer assigned
- [ ] First week training schedule created

**New Employee Receives:**
- [ ] Welcome email with CRM login credentials
- [ ] Link to user manual
- [ ] Link to video tutorials
- [ ] Printed quick reference cheatsheets
- [ ] Training schedule
- [ ] Contact info for trainer/mentor

## DAY 1: SYSTEM BASICS

### Morning: Introduction & Navigation (2 hours)

**System Overview** ✓ Manager presents
- [ ] Read "Getting Started" section in User Manual
- [ ] Watch introduction video (if available)
- [ ] Understand CRM purpose and benefits
- [ ] Learn about different user roles

**Login & Account Setup** ✓ Hands-on
- [ ] Successfully log into CRM
- [ ] Change password (first login)
- [ ] Set up two-factor authentication (if required)
- [ ] Bookmark CRM URL
- [ ] Verify email notifications are working

**Navigation Training** ✓ Hands-on
- [ ] Identify and use main navigation sidebar
- [ ] Locate dashboard and understand widgets
- [ ] Use search functionality
- [ ] Open and close dialogs/modals
- [ ] Practice keyboard shortcuts (Ctrl+K, Esc, etc.)
- [ ] Understand breadcrumb navigation

**Understanding Your Role** ✓ Discussion with manager
- [ ] Review your specific permissions
- [ ] Understand what you can/cannot do
- [ ] Identify features relevant to your role
- [ ] Know who to escalate to for different issues

### Afternoon: Dashboard & Basic Features (2 hours)

**Dashboard Walkthrough** ✓ Hands-on
- [ ] Understand each dashboard widget
- [ ] Read quick stats (revenue, jobs, opportunities, reviews)
- [ ] Review activity feed
- [ ] Check pending actions
- [ ] Navigate to different sections from dashboard

**Customer Lookup** ✓ Practice exercises
- [ ] Search for customer by name
- [ ] Search for customer by phone number
- [ ] Search for customer by email
- [ ] View customer profile
- [ ] Understand customer information displayed
- [ ] Navigate customer history

**Practice Exercises**
- [ ] Find 5 specific customers by name
- [ ] Look up customer by phone: 555-1234 (example)
- [ ] Locate customer with most recent service
- [ ] Find customer in specific zip code

**End of Day 1**
- [ ] Quiz: System navigation (5 questions)
- [ ] Confirm understanding of role and permissions
- [ ] Review any questions or concerns
- [ ] Preview Day 2 topics

- [ ] Assign opportunity to team member

- [ ] Navigate opportunity board
- [ ] Drag opportunity between stages

**Follow-up & Conversion** ✓ Hands-on practice
