## Common Questions

**Q: Do all customers need to register?**
A: No, it's optional, but highly beneficial. Encourage but don't require.

**Q: Can customers book without registering?**
A: Depends on your settings. You can allow guest booking or require registration.

**Q: Is customer data secure?**
A: Yes, portal uses bank-level encryption and follows all security best practices.

**Q: Can I limit which customers have access?**
A: Yes, you can enable/disable portal access for individual customers if needed.

**Q: What if customer forgets password?**
A: Self-service password reset via email. They can also request magic link (passwordless login).
