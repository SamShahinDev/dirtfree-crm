# Video Tutorial Scripts

This directory contains scripts for producing training videos for the Dirt Free CRM system.

## Available Tutorials

1. **opportunities-overview.md** - Complete guide to opportunities management (8-10 minutes)
2. **creating-promotions.md** - How to create and send promotional campaigns (6-8 minutes)
3. **review-management.md** - Managing reviews and automating collection (5-7 minutes)
4. **loyalty-program.md** - Understanding and managing the loyalty program (7-9 minutes)
5. **analytics-dashboard.md** - Navigating analytics and reports (6-8 minutes)
6. **customer-portal.md** - Customer portal features and settings (5-6 minutes)

## Production Guidelines

### Video Format
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30fps
- **Duration**: 5-10 minutes per tutorial
- **Format**: MP4 (H.264 codec)

### Screen Recording
- Use screen capture software (Loom, Camtasia, ScreenFlow)
- Record at native resolution
- Use cursor highlighting
- Zoom in on important UI elements
- Clear, high-quality audio

### Voiceover Guidelines
- Professional, friendly tone
- Clear enunciation
- Moderate speaking pace (not too fast)
- Background music at low volume (optional)
- Remove "ums" and long pauses

### On-Screen Elements
- Highlight cursor clicks
- Add text callouts for key points
- Use zoom/pan for important details
- Include keyboard shortcut overlays
- Add chapter markers

### Branding
- Include Dirt Free logo in corner
- Use brand colors for highlights
- Add intro/outro (3-5 seconds each)
- Include contact information in outro

## File Naming Convention

Use this format: `[topic]-[version].mp4`

Examples:
- `opportunities-overview-v1.mp4`
- `creating-promotions-v2.mp4`

## Publishing

Once produced, videos should be:
1. Uploaded to company YouTube channel (unlisted)
2. Embedded in in-app help system
3. Linked from user manual
4. Included in onboarding emails
5. Added to knowledge base

## Updating Videos

When features change:
1. Note timestamp of changed section
2. Record new segment
3. Edit into existing video OR
4. Create new version and archive old one
5. Update all links and references

## Translation

For multi-language support:
1. Create subtitle files (.srt)
2. Translate scripts
3. Record voiceovers in other languages
4. Upload multiple language versions

## Accessibility

Ensure all videos include:
- Closed captions (auto-generated + manually reviewed)
- Audio descriptions for visual elements
- Transcripts available as text
- Keyboard navigation demonstrations
